# MachineLeaningBook
This repository keeps code examples and exercices of the book "Hand's on: Machine Leaning with Scikit-Learn, Keras and TensorFlow"

I'm using a virtual enviroment that manages the python modules, i strongly recommmends that you use a venv and not install your modules globally.
If you don't now how to use a venv [here](https://docs.python.org/pt-br/3/library/venv.html) is the venv docummentation.
